# 基于aba热门搜索词报告的搜索词精准投放拓词规则
## aba热门搜索词报告
### 字段
- 搜索词（Search Term）
- 点击量（Clicks）
- 转化量（Orders）
- 点击份额（Click Share）
- 转化份额（Conversion Share）
- 出单效率（Efficiency = Conversion Share / Click Share）
- Top ASIN占比（Top ASIN Click/Conversion Concentration）
- 曝光量（Impressions）
- 广告位（Placement）

## 拓词规则
| 点击份额 | 转化份额 | 出单效率 | 策略 |
|-----------|-----------|-----------|------|
| 高        | 高        | 高        | 产品有优势时全力打搜索结果顶部 |
| 高        | 高        | 低        | 打搜索结果顶部 + 其余位置，优先其余位置，放量时可拓展顶部 |
| 高        | 低        | 高        | 自相矛盾，不存在这种情况 |
| 高        | 低        | 低        | 只打搜索结果其余位置，放量时可测试顶部数据 |
| 低        | 高        | 高        | 只打搜索结果顶部，可尝试提高竞价尝试能否提高点击份额 |
| 低        | 高        | 低        | 自相矛盾，不存在这种情况 |
| 低        | 低        | 高        | 打搜索结果顶部 + 其余位置，优先顶部 |
| 低        | 低        | 低        | 词较泛，放弃，放量时可测试顶部和其余位置 |

## 简化版本
| 点击份额 | 转化份额 | 出单效率 | 策略 |
|-----------|-----------|-----------|------|
| 高        | 高        | 高        | 搜索结果顶部 |
| 低        | 高        | 高        | 搜索结果顶部 |
| 低        | 低        | 高        | 打搜索结果顶部 + 其余位置，优先顶部 |
| 高        | 高        | 低        | 打搜索结果顶部 + 其余位置，优先其余位置 |
| 高        | 低        | 低        | 搜索结果其余位置 |

## 分类阈值
- 点击份额：20%
- 转化份额：16%
- 出单效率：0.8 （点击份额阈值 / 转化份额阈值）
