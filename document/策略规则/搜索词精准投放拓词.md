# 基于aba热门搜索词报告的搜索词精准投放拓词规则
## aba热门搜索词报告
热门搜索词报告有日、周、月以及季维度
报告收集了用户在亚马逊有进行搜索的搜索词，按照搜索次数降序排序
日维度收录一日有搜索至少1次的词，周维度收录一周至少7次，月维度收录一月至少30次，季维度收录一季至少90次
每一条包含了搜索词排名，前三商品点击份额，前三商品转化份额，前三商品信息

## 拓词规则
| 点击份额 | 转化份额 | 出单效率 | 策略 |
|-----------|-----------|-----------|------|
| 高        | 高        | 高        | 产品有优势时全力打搜索结果顶部 |
| 高        | 高        | 低        | 打搜索结果顶部 + 其余位置，优先其余位置，放量时可拓展顶部 |
| 高        | 低        | 高        | 自相矛盾，不存在这种情况 |
| 高        | 低        | 低        | 只打搜索结果其余位置，放量时可测试顶部数据 |
| 低        | 高        | 高        | 只打搜索结果顶部，可尝试提高竞价尝试能否提高点击份额 |
| 低        | 高        | 低        | 自相矛盾，不存在这种情况 |
| 低        | 低        | 高        | 打搜索结果顶部 + 其余位置，优先顶部 |
| 低        | 低        | 低        | 词较泛，放弃，放量时可测试顶部和其余位置 |

## 分类阈值
- 点击份额：20%
- 转化份额：16%
- 出单效率：0.8 （点击份额阈值 / 转化份额阈值）

## 数据好的商品定义
在“词 -> ASIN -> 词”链路中，数据好的商品定义为：

- 在历史窗口内至少出现 2 周
- 加权出单效率 >= 0.80

其中：
- 商品历史轨迹来自该搜索词历史周数据中的前三商品
- 商品点击份额 / 转化份额取对应 `top_product_x_*share`
- 时间加权采用指数衰减

如果没有任何商品满足以上条件，则启用兜底规则：

- 只返回 1 个商品 ASIN
- 兜底候选范围只取每周点击排名第一的商品
- 在兜底候选中，优先选择加权出单效率最高的商品
- 若加权出单效率并列，则优先选择加权点击份额更高的商品

## 拓词强度
当前实现支持三档拓词强度：

### conservative
- 排名保留比例：30%

### balanced
- 排名保留比例：50%

### aggressive
- 排名保留比例：80%

说明：
- 排名截断发生在候选词汇总、去重、相关性判断之后
- 当前按候选池内的 `search_rank` 升序排序，再按拓词强度保留前若干百分比
